What is this?

I love Amazon's wishlists and also love deals. "Sort by Price" on Amazon's wishlist page now is very very broken, so I built this to help everyone find the best deals on things to buy me.

Where does the data come from?

I use the Amazon Product Advertising API to get the information on indvidual products, using bottlenose as a Python wrapper for the API.
They don't offer an API to get wishlists, so I scrape those using requests + BeautifulSoup. (It works really well until Amazon realizes it's a script and not a human, so I'm usually not able to pull the full wishlist each day).

Couldn't this have been generalized to check wishlists for anybody?

Maybe, but the Amazon Product Advertising API is a) highly contstrained on the number of calls/minute you can make AND b) requires a lot of calls to get all the information we want.
